# React Data Management Example App

This example application is intended to serve as an illustration for the React Data Management article in [Smashing Magazine](???).

## Technologies Overview

There are multiple 3rd party tools and npm modules, but the main backbone of the application uses React, Redux over ES6.

## Purpose

The main purpose of the article, and this accompanying set of applications (via various branches in this repository) is to demonstrate modern techniques of data management in a React (and Redux) application.
